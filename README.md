# friendbuy


### Prerequisites
- [node](https://nodejs.org/en/download/)
- run in bash/zsh

### Running the program
- `git clone https://github.com/chrisbendel/friendbuy && cd friendbuy`
- `npm install`
- `node ./main.js` or `npm run start`
- To run test suite: run `npm run test`
